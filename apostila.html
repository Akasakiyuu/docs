Apostila HTML e CSS

Introdução ao HTML e CSS

HTML

  • HTML - Hipertext Markup Language
  • Liguaguem de programação ultilizada para criação de paginas web
  • Objetivo de formatar textos através de marcações especiais denominadas tags, para que possam ser exibidos de forma conveniente pelos clientes
  • Possibilita a interligação entre páginas da Web, criando assim documentos com o conceito de hipertexto

CSS

  • Linguagem ultiliazada para estilizar a pag html criada
  • Podendo definir cores, tamanho, fundo da web

Tags HTML

As tags no HTML são palavras que ficam entre os sinais de de menor e maior

  • html - identifica o documento como um hipertexto htm
  • head - meta dado, dentro do head o usuario n tem visao, onde css estara
  • title - é onde vai definir o nome da aba q estara no navegador
  • body - o usuario tem acesso, é o corpo, conteudo da pagina
  • p - paragrafo, onde todo o texto ficara
  • h1 - sao tags em blocos e por isso o a linha é quebrada, sempre que declarados h1, h2 serao novos blocos e por isso sempre terá a quebra de linha
  • span - para coisas importantes
  • strike - é dentro do paragrafo onde a função dela é riscar a frase, colocar uma linha por cima do texto
  • sub - sub escrito ou seja vai ser o texto mais baixo que o resto
  • sup - Texto mais a cima do resto
  • big e small - deixa o texto maior ou menor do que o resto
  • tt - Uma fonte com espaçamento
  • pre - Uma formataão literal, ler o enter/tab, comando que o html n interpretaria
  • cite - iatlico
  • strong - negrito
  • em - da enfase na palavra/texto
  • blockquote - citação longa de origem de outro site
  • q - citação menor que blockquote

CSS

CSS é chamado de linguagem Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. Pense na decoração da sua página. Utilizando o CSS é possível alterar a cor do texto e do fundo, fonte e espaçamento entre parágrafos. Também pode criar tabelas, usar variações de layouts, ajustar imagens para suas respectivas telas e assim por diante.
